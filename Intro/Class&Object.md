# Class & Object

## Class

It is defined data type blueprint for creating objects. It can have attributes and methods. It is a user defined data type. It is a template for creating objects. 

## Object

It is an instance of a class. It is a real world entity. It is a variable of a class. 

## Example

```
class Student{
    public:
        string name;
        int age;
        int roll;
        string grade;
};
```

### Object

```
Student s1;
```

### Accessing the members of the class

```
s1.name = "Tasrik";
s1.age = 20;
s1.roll = 1;
s1.grade = "A";
```
